## Overview

SmartGarden-AI leverages AI algorithms to analyze environmental data collected from IoT sensors placed in your garden. This system helps optimize plant care by automating watering and providing actionable insights to enhance plant growth and health.

---

## Features

- Real-time monitoring of soil moisture, temperature, and light levels
- AI-driven watering automation based on plant needs
- Historical data visualization and trend analysis
- User-friendly web dashboard
- Mobile notifications and alerts

---

## Tech Stack

- **Frontend:** React.js
- **Backend:** Node.js, Express
- **AI Models:** TensorFlow.js
- **Database:** MongoDB
- **IoT Integration:** MQTT protocol
- **Deployment:** Docker, AWS

## Installation

1. Clone the repository:

   ```bash
   git clone https://github.com/dianebaeza/SmartGarden-AI.git
   cd SmartGarden-AI
   ```

2. Install backend dependencies:

   ```bash
   cd backend
   npm install
   ```

3. Install frontend dependencies:

   ```bash
   cd ../frontend
   npm install
   ```

4. Configure environment variables for database and MQTT broker.

5. Start backend server:

   ```bash
   cd ../backend
   npm start
   ```

6. Start frontend:

   ```bash
   cd ../frontend
   npm start
   ```

Ensure your IoT sensors are connected and configured to send data to the MQTT broker.

## Usage

- Access the web dashboard at `http://localhost:3000`.
